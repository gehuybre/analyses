# Blog Repo Split Migration

This folder contains a concrete migration plan, structure guidance, and templates for splitting blog content into a new repo while keeping this repo as a hub of title pages + links.

## Contents
- MIGRATION_PLAN.md: phased plan with entry/exit criteria and risks.
- REPO_STRUCTURE.md: proposed hub + blogs repo structure.
- TODO.md: actionable checklist for the migration.
- MIGRATION_STATUS.md: table to track progress per blog.
- PLAN_STRUCTURE_TEMPLATE.md: reusable plan structure for future migrations.
- templates/: starter files for manifests and CI workflows.

## How to use
1. Review MIGRATION_PLAN.md and adjust scope/timing.
2. Fill in MIGRATION_STATUS.md with all current blogs.
3. Customize templates and copy into the target repos.
4. Track tasks in TODO.md as you migrate.
